# Pixel Avatar Customization

## 배경
KeyWork는 레트로(Win98/Mac Classic) 테마 기반 타이핑 연습 웹앱으로, 현재 프로필 아바타가 닉네임 첫 글자만 표시하는 단순한 형태임. 사용자 개성 표현과 멀티플레이 대전 몰입감을 위해 도트(8-bit) 스타일 파츠 조합형 아바타 시스템을 도입한다.

## 목적
- 사용자가 머리/눈/입/모자/악세서리 등의 파츠를 조합하여 고유한 도트 아바타를 생성
- 프로필, 사이드바, 멀티 대전, 리더보드 등 앱 전반에서 캐릭터 표시
- 레트로 테마(win98 bevel, mac-classic)와 자연스러운 시각적 통합

## 범위
- 파츠 기반 도트 아바타 에디터 UI
- 캐릭터 데이터 저장 (Supabase profiles 확장)
- 앱 전역 아바타 렌더링 컴포넌트
- 멀티플레이 대전 시 상대 아바타 표시

## 제외 범위
- 아바타 파츠 유료 판매 / 잠금 해제 시스템
- 아바타 애니메이션 (v1은 정적 이미지)
- 아바타 이미지 파일 업로드 (커스텀 이미지 업로드 X, 파츠 조합만)
