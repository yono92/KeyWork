# spec.md

## 사용자 스토리
- 플레이어로서 한글 모드에서 한글 중심 문장만 보고 싶다.

## 기능 요구사항
1. 문장 반영 전에 언어별 허용 문자 집합으로 정규화한다.
2. 한글 모드: 한글/숫자/공백/기본 문장부호만 허용한다.
3. 영어 모드: 영문/숫자/공백/기본 문장부호만 허용한다.
4. 정규화 결과가 비면 폴백 문장으로 대체한다.

## Acceptance Criteria
1. 한글 모드에서 A-Z 문자가 제거된다.
2. 이모지/특수기호가 제거된다.
3. 기본 문장부호(`.,!?"'():;-`)와 공백은 유지된다.
4. lint/test/build 통과.
