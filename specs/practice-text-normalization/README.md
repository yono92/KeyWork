# README.md

## 기능 배경
- 한글 연습 모드에서 영어/특수문자가 섞여 입력 난이도와 일관성이 떨어지는 문제가 있다.

## 목적
- 문장 노출 전에 문자 집합을 정규화해 기본 키보드로 처리 가능한 범위만 유지한다.

## 범위
- `TypingInput`의 위키/속담 문장 반영 경로
- `sentenceUtils` 문장 정규화 유틸

## 제외 범위
- 다른 게임 모드의 단어 데이터 정제
- 키보드 컴포넌트 레이아웃 변경

## 성공 기준
- 한글 모드에서 영어 문자가 노출되지 않는다.
- 과도한 특수문자가 제거된다.
- lint/test/build 통과.
