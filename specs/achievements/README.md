# 업적/뱃지 시스템 (Achievements)

## 배경

유저 전적 페이지(F3)로 통계는 보여주고 있지만, 목표 달성에 대한 보상이나 동기 부여 요소가 없음.
게임 기록이 쌓여도 "다음 목표"가 없어 재방문 유인이 약함.

## 목적

게임 플레이 기록 기반으로 업적을 자동 해금하여, 프로필에 뱃지로 표시.
게임 오버 시 새로 해금된 업적을 알림으로 보여줌.

## 범위

- 업적 정의 테이블 + 유저별 해금 테이블 (Supabase)
- 점수 제출 후 업적 조건 검사 (클라이언트)
- 프로필 페이지에 업적 섹션 추가
- 게임 오버 시 신규 업적 팝업
- 초기 업적 20~30개 정의

## 제외 범위

- 업적 진행도 바 (해금/미해금만, 부분 진행 없음)
- 업적별 보상 (스킨, 아이템 등 — 향후 확장 가능)
- 관리자 업적 편집 UI
