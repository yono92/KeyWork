# spec.md

## 문제
현재 단어 다양성 로직은 공유 접두 길이 중심이라 `사과 -> 사과즙`, `학교 -> 학교생활` 같은 접두 확장 패턴을 충분히 막지 못한다.

## 목표
1. 최근 단어에 접두/포함 형태로 이어 붙는 후보를 더 강하게 회피한다.
2. 작은 데이터 풀에서도 게임이 멈추지 않도록 폴백은 유지한다.
3. 기존 게임 로직 변경 없이 출제 정책만 강화한다.

## 수용 기준
1. 최근 단어와의 접두/포함 관계를 한국어 우선으로 추가 회피한다.
2. 최근 단어 이력이 기존보다 길게 유지된다.
3. `npm run lint`, `npm run test:run`, `npm run build` 통과
