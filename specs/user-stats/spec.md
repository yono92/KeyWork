# 유저 전적/통계 — Spec

## 사용자 스토리

1. 로그인한 유저가 프로필 페이지에서 "내 전적" 섹션을 볼 수 있다.
2. 게임 모드별 최고 점수, 플레이 횟수, 평균 정확도가 카드로 표시된다.
3. 최근 10개 매치 기록이 리스트로 표시된다.
4. 멀티플레이 전적(승/패/승률)이 요약된다.

## 기능 요구사항

### FR-1: 모드별 통계 카드
- 각 게임 모드별: 최고 점수, 플레이 횟수, 평균 정확도(있는 경우)
- 게임 모드: practice, falling-words, word-chain, typing-defense, dictation, tetris
- 기록이 없는 모드는 "아직 기록 없음" 표시

### FR-2: 종합 통계
- 총 플레이 횟수
- 멀티플레이 승률 (승/패/전체)
- 가장 많이 플레이한 모드

### FR-3: 최근 매치 기록
- 최근 10건의 game_scores 레코드
- 표시: 게임 모드, 점수, 정확도(있으면), 날짜
- 멀티플레이의 경우 승/패 표시

### FR-4: 데이터 조회
- Supabase에서 직접 쿼리 (클라이언트 사이드)
- game_scores 테이블의 기존 인덱스 활용 (idx_scores_user)
- 로그인하지 않은 경우 "로그인 필요" 안내

## Acceptance Criteria

- [ ] AC-1: 프로필 페이지에 통계 섹션 표시
- [ ] AC-2: 모드별 최고 점수, 플레이 횟수, 평균 정확도 표시
- [ ] AC-3: 최근 10건 매치 기록 표시
- [ ] AC-4: 멀티플레이 승률 표시
- [ ] AC-5: 비로그인 시 안내 메시지
- [ ] AC-6: 레트로 테마 적용
