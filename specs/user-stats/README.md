# 유저 전적/통계 페이지 (User Stats)

## 배경

game_scores 테이블에 데이터가 쌓이고 있지만, 유저별 통계를 보여주는 UI가 없음.
프로필 페이지는 계정/아바타 관리만 제공.

## 목적

프로필 페이지에 게임별 통계 섹션을 추가하여 유저의 전적, 최고 기록, 플레이 횟수 등을 시각화.

## 범위

- 프로필 페이지에 통계 섹션 추가
- 게임 모드별 최고 점수, 플레이 횟수, 평균 정확도
- 최근 매치 기록 (10건)
- 멀티플레이 승률
- Supabase RPC 함수로 서버사이드 집계

## 제외 범위

- 추이 그래프 (차트 라이브러리 추가 없이, 숫자 통계만)
- 다른 유저의 통계 조회 (본인만)
