# spec.md

## 문제
1. 현재 앱은 플레이 기록은 쌓이지만 "오늘 다시 들어와야 하는 이유"가 약하다.
2. 전적/업적만으로는 장기 재방문을 만들기 어렵고, 짧은 세션 보상이 부족하다.

## 목표
1. 프로필에서 오늘의 미션 진행 상황을 바로 볼 수 있게 한다.
2. 최근 연속 플레이 스트릭을 시각적으로 보여 줘 재방문 동기를 만든다.
3. 기존 `game_scores` 데이터만으로 동작해 빠르게 출시 가능한 리텐션 루프를 만든다.

## 비목표
- 미션 보상 지급 시스템
- 푸시 알림/이메일 리마인더
- 별도 DB 테이블 추가

## 수용 기준
1. 프로필에서 오늘의 미션 3개와 진행 상태를 볼 수 있다.
2. 현재 스트릭, 최고 스트릭, 최근 7일 활동 상태를 볼 수 있다.
3. 미션/스트릭은 `game_scores` 데이터만으로 계산된다.
4. 로그인하지 않은 경우 기존 프로필 접근 정책을 유지한다.
5. `npm run lint`, `npm run test:run`, `npm run build`가 통과한다.
