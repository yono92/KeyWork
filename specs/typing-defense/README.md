# Typing Defense (타이핑 디펜스)

## 배경

KeyWork는 6가지 게임 모드를 계획했으나, 타이핑 디펜스와 받아쓰기가 미구현 상태.
CLAUDE.md에 게임 설명이 명시되어 있고, Supabase `game_scores` 타입에 `typing-defense` 모드가 이미 정의되어 있음.

## 목적

타워 디펜스 장르와 타이핑 연습을 결합한 아케이드 게임 모드 추가.
적이 기지를 향해 행진 → 단어를 타이핑해서 처치 → 웨이브 클리어 → 보스전 구조.

## 범위

- 싱글 레인 타워 디펜스 (적이 오른쪽→왼쪽 이동)
- 웨이브 시스템 (5웨이브 + 보스)
- 난이도 선택 (Easy/Normal/Hard)
- 콤보/점수/목숨 시스템
- 리더보드 연동
- 한/영 지원

## 제외 범위

- 멀티플레이어 (추후 확장)
- 다중 레인
- 타워 건설/업그레이드 (순수 타이핑 기반)
