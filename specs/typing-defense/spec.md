# Typing Defense — Spec

## 사용자 스토리

1. **타이핑 디펜스 시작**: 사용자가 사이드 네비에서 "타이핑 디펜스"를 선택하면 난이도 선택 화면이 표시된다.
2. **난이도 선택**: Easy/Normal/Hard 중 선택하면 3초 카운트다운 후 게임 시작.
3. **적 처치**: 화면 오른쪽에서 적이 등장하며 머리 위에 단어가 표시됨. 해당 단어를 입력창에 타이핑하면 적이 처치됨.
4. **기지 방어**: 적이 왼쪽 기지에 도달하면 목숨 1개 소모. 목숨 0이면 게임 오버.
5. **웨이브 진행**: 5웨이브마다 보스 등장. 보스는 긴 문장을 타이핑해야 처치. 웨이브 클리어 시 짧은 휴식.
6. **콤보**: 연속 처치 시 콤보 배율 상승 → 점수 증가. 놓치면 콤보 리셋.
7. **게임 오버**: 최종 점수, 웨이브 수, 처치 수, 최대 콤보 표시. 리더보드 제출 (로그인 시).
8. **일시정지**: ESC로 일시정지/재개.

## 기능 요구사항

### FR-1: 게임 필드
- 가로 스크롤 싱글 레인
- 왼쪽: 기지 (방어 대상), 오른쪽: 적 스폰 지점
- 레트로 픽셀 스타일 (기존 테마 활용)
- 기지 HP 바 표시

### FR-2: 적 (Enemy) 시스템
- 일반 적: 단어 1개 (word.json에서 선택)
- 빠른 적: 짧은 단어, 이동 속도 빠름
- 탱크 적: 긴 단어, 이동 속도 느림
- 보스: 속담/문장 (proverbs.json), 큰 크기, HP 바

### FR-3: 웨이브 시스템
- 웨이브당 적 수: `기본 5 + (웨이브 * 2)`, 최대 20
- 5웨이브마다 보스 웨이브 (일반 적 + 보스 1마리)
- 웨이브 간 3초 휴식 (다음 웨이브 번호 표시)
- 무한 웨이브 (게임 오버까지 계속)

### FR-4: 타이핑 입력
- 화면 하단 고정 입력창
- 한국어 IME 지원 (GameInput 컴포넌트 재사용)
- Enter로 제출, 매칭되는 적 자동 탐색
- 부분 매칭 하이라이트 (현재 입력과 일치하는 적 표시)
- 여러 적이 같은 단어면 가장 앞(왼쪽)에 있는 적 우선 매칭

### FR-5: 점수 시스템
- 기본 점수: `글자수 × 10`
- 콤보 배율: `min(1 + combo × 0.2, 3.0)` (최대 3배)
- 보스 처치 보너스: `× 5`
- 웨이브 클리어 보너스: `웨이브 번호 × 100`
- 난이도 배율: Easy 0.7×, Normal 1.0×, Hard 1.5×

### FR-6: 난이도
| 항목 | Easy | Normal | Hard |
|------|------|--------|------|
| 적 이동 속도 | 0.6× | 1.0× | 1.4× |
| 스폰 간격 | 1.5× | 1.0× | 0.7× |
| 목숨 | 5 | 3 | 2 |
| 점수 배율 | 0.7× | 1.0× | 1.5× |

### FR-7: 시각 효과
- 적 처치: 폭발 파티클 + 점수 팝업
- 보스 처치: 큰 폭발 + 화면 흔들림
- 콤보 표시 (×2, ×3 등)
- 적 기지 도달 시: 화면 붉은 플래시
- 웨이브 클리어: 축하 이펙트
- FX ON/OFF 토글 지원

### FR-8: 사운드
- useGameAudio 재사용
- 처치: "destroy" / 놓침: "lifeLost" / 보스 처치: "win"
- 웨이브 클리어: "roundComplete" / 게임오버: "gameOver"

## 비기능 요구사항

- **NFR-1**: 60fps 유지 (requestAnimationFrame 기반 이동)
- **NFR-2**: 모바일 대응 (세로 레이아웃, 터치 입력)
- **NFR-3**: 기존 코드 패턴 준수 (FallingWordsGame, TypingRunnerGame과 동일 구조)
- **NFR-4**: 접근성 (aria-live 스코어, 키보드 전용 플레이)

## Acceptance Criteria

- [ ] AC-1: 난이도 선택 후 3초 카운트다운 → 게임 시작
- [ ] AC-2: 적이 오른쪽에서 왼쪽으로 이동하며, 단어 타이핑으로 처치 가능
- [ ] AC-3: 적이 기지 도달 시 목숨 감소, 0이면 게임 오버
- [ ] AC-4: 5웨이브마다 보스 등장, 문장 타이핑으로 처치
- [ ] AC-5: 콤보 시스템 동작 (연속 처치 → 배율 증가)
- [ ] AC-6: 게임 오버 시 GameOverModal 표시 + 리더보드 제출
- [ ] AC-7: ESC로 일시정지/재개
- [ ] AC-8: 한국어/영어 전환 시 적 단어도 해당 언어로 변경
- [ ] AC-9: 사이드 네비에서 타이핑 디펜스 항목 표시 및 라우팅
- [ ] AC-10: 레트로 테마(win98/mac-classic) 적용
