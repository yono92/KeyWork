# spec.md

## 문제
1. 현재 친구 기능은 추가/수락/삭제까지만 있고, 친구에게 바로 대전을 거는 액션이 없다.
2. 게임 초대 토스트는 멀티플레이 로비 안에서만 보여서, 다른 화면에 있으면 초대를 놓친다.
3. 경쟁 서비스 방향이라면 친구에게 도전하고 즉시 수락/입장하는 루프가 필요하다.

## 목표
1. 프로필 친구 목록에서 게임 모드를 선택해 바로 도전장을 보낼 수 있게 한다.
2. 전역 초대 토스트를 통해 로비 밖에서도 초대를 수락할 수 있게 한다.
3. 초대 수락 시 해당 게임의 온라인 화면으로 즉시 이동하고 방 참가까지 이어지게 한다.

## 비목표
- 새 친구 테이블/초대 테이블 추가
- 채팅 기능
- 친구 전용 랭킹

## 수용 기준
1. 친구 목록에서 `도전` 액션을 사용할 수 있다.
2. 전역 초대 토스트가 게임/프로필/랭킹 화면 어디서든 보인다.
3. 초대 수락 시 해당 게임 온라인 모드로 이동해 초대된 방에 입장한다.
4. 기존 로비 내 친구 초대 흐름은 깨지지 않는다.
5. `npm run lint`, `npm run test:run`, `npm run build`가 통과한다.
