# spec.md

## 문제
1. 현재 로비는 빠른 매칭, 코드 참가, 대기방 목록이 동시에 보여 사용자가 메인 흐름을 이해하기 어렵다.
2. 사용자는 공개 방 목록을 보고 참가하는 단일 방식으로 단순화하길 원한다.

## 목표
1. 공개 방 생성과 목록 참가만 남겨 온라인 대전 진입 흐름을 하나로 정리한다.
2. 대기 중 화면에서도 방 코드 대신 공개 대기방 상태를 중심으로 안내한다.

## 비목표
- 매치메이킹 규칙 변경
- Realtime 동기화 방식 변경
- 게임 규칙 변경

## 수용 기준
1. 로비 메인 화면에서 `빠른 매칭`, `코드로 참가`, 방 코드 입력 UI가 제거된다.
2. `방 만들기`는 공개 방 생성 의미로 보이고, 다른 사용자는 목록에서만 참가한다.
3. 대기 상태 UI가 목록형 공개 로비와 어울리는 문구로 갱신된다.
4. `npm run lint`, `npm run test:run`, `npm run build`가 통과한다.
