# 친구 시스템 & 초대 (Friends & Invites)

## 배경

멀티플레이는 공개 방 목록에서 익명 매칭만 가능. 특정 유저와 반복 대전하거나 초대하는 방법이 없음.

## 목적

친구 추가/관리 + 온라인 친구에게 대전 초대를 보내는 기능 추가.
프로필 페이지에 친구 목록, 멀티플레이 로비에 친구 초대 버튼.

## 범위

- friendships 테이블 (친구 요청/수락/거절)
- 닉네임 검색으로 친구 요청
- 프로필 페이지 친구 섹션 (목록, 요청 관리)
- 멀티플레이 로비에서 친구 초대 (Realtime broadcast)
- 온라인 상태 표시 (Realtime presence)

## 제외 범위

- 차단 기능
- 친구 수 제한
- 채팅
