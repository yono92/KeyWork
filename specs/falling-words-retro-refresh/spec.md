# spec.md

## 사용자 스토리
- 플레이어로서 단어낙하 모드도 다른 게임처럼 레트로 감성으로 플레이하고 싶다.

## 기능 요구사항
1. 메인 게임 프레임과 HUD를 레트로 보더/팔레트로 표현한다.
2. 단어와 점수 표기를 픽셀 느낌(모노/하드 섀도우)으로 보강한다.
3. 시작 오버레이와 하단 입력 패널도 동일한 레트로 패턴을 적용한다.
4. 기존 상호작용(입력, 점수, 콤보, 게임오버)은 변경하지 않는다.

## Acceptance Criteria
1. FallingWords 화면에서 기존 대비 레트로 톤이 명확하게 강화된다.
2. 게임 플레이 기능은 동일하게 동작한다.
3. lint/test/build 통과.
