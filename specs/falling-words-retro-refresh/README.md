# README.md

## 기능 배경
- 단어낙하 모드가 다른 모드 대비 시각적으로 모던하게 보여 레트로 톤 일관성이 떨어진다.

## 목적
- 기존 게임 동작을 유지하면서 단어낙하 화면을 레트로 UI 톤으로 강화한다.

## 범위
- `src/components/FallingWordsGame.tsx` 시각 스타일 조정

## 제외 범위
- 점수/난이도/아이템 로직 변경
- 애니메이션 키프레임 대규모 개편

## 성공 기준
- 화면 프레임, 상단 HUD, 단어 렌더링, 시작 오버레이가 레트로 스타일로 통일된다.
- 기능 회귀 없이 lint/test/build 통과.
