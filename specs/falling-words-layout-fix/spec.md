# spec.md

## 사용자 스토리
- 플레이어로서 단어낙하 모드가 데스크톱과 태블릿 폭에서도 정돈된 프레임으로 보여야 한다.
- 플레이어로서 좌측 네브의 설정 버튼이 겹치지 않아야 빠르게 환경을 바꿀 수 있다.

## 기능 요구사항
1. 단어낙하 게임판은 중간 폭 데스크톱에서도 시작 오버레이와 HUD가 과도하게 작아 보이거나 비정상적으로 배치되지 않아야 한다.
2. 좌측 사이드 네브의 하단 컨트롤 트레이는 `md` 구간의 좁은 폭에서도 내부 버튼이 넘치지 않아야 한다.
3. 테마/언어/사운드/FX/다크모드 토글은 기존 기능을 유지해야 한다.
4. 단어낙하 게임 로직과 키 입력 흐름은 변경하지 않는다.

## Acceptance Criteria
1. `falling-words` 화면에서 시작 오버레이와 게임판이 중간 폭 데스크톱에서도 읽기 쉬운 비율로 보인다.
2. `SideNav` 하단 설정 영역이 `md` 구간에서 옆 패널과 겹치지 않는다.
3. 기존 정적 점검, 테스트, 빌드가 통과한다.
