# spec.md

## 기능명
모바일 경량 테트리스 모드 추가

## 문제 정의 (사용자 관점)
- 현재 노출된 모드가 제한적이며, 타이핑 외 가벼운 콘텐츠 선택지가 부족하다.
- 모바일에서 즉시 조작 가능한 퍼즐 모드가 없어 짧은 플레이 동기가 약하다.

## 사용자 스토리
1. 사용자는 모바일에서 터치 버튼만으로 간단히 테트리스를 플레이하고 싶다.
2. 사용자는 데스크톱에서 방향키로 동일하게 플레이하고 싶다.
3. 사용자는 별도 설정 없이 시작/일시정지/재시작을 직관적으로 제어하고 싶다.

## 기능 요구사항
1. `/tetris` 라우트에서 즉시 플레이 가능한 보드를 제공해야 한다.
2. 좌/우 이동, 회전, 소프트 드롭, 하드 드롭 조작을 지원해야 한다.
3. 시작/일시정지/새 게임 버튼을 제공해야 한다.
4. 점수/라인/레벨 상태를 실시간 표시해야 한다.
5. 게임오버 상태를 명확히 표시하고 재시작이 가능해야 한다.
6. 랜딩 페이지와 사이드 메뉴에서 테트리스 모드 진입이 가능해야 한다.

## 비기능 요구사항
1. 외부 의존성 추가 없이 React 상태 기반으로 동작해야 한다.
2. 모바일 화면(폭 360px 기준)에서 조작 버튼이 가려지지 않아야 한다.
3. 기존 모드 동작/라우팅을 깨지 않아야 한다.

## 수용 기준 (Acceptance Criteria)
1. 홈에서 "테트리스" 카드를 눌러 `/tetris`로 이동할 수 있다.
2. 사이드 메뉴에서 "Tetris" 버튼으로 이동할 수 있다.
3. 게임 시작 후 블록이 자동 하강하고, 라인 제거 시 점수/라인이 증가한다.
4. 블록 스폰 불가 시 게임오버가 표시되고 새 게임으로 재시작 가능하다.
5. `npm run lint`, `npm run test:run`, `npm run build`가 통과한다.
