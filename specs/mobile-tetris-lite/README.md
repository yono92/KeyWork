# 모바일 경량 테트리스 (mobile-tetris-lite)

## 개요
기존 KeyWork의 가벼운 게임 철학을 유지하면서, 모바일 웹에서도 즉시 플레이 가능한 단순 테트리스 모드를 추가한다.

## 목적
- 타이핑 중심 모드 외에 짧게 즐길 수 있는 리프레시 게임을 제공한다.
- 별도 라이브러리 없이 경량 구현으로 초기 로딩 부담을 늘리지 않는다.
- 터치 조작과 키보드 조작을 동시에 지원해 모바일/데스크톱 접근성을 확보한다.

## 범위
- 신규 게임 모드 `tetris` 추가
- 랜딩/사이드 메뉴 노출
- 기본 점수/라인/레벨 기반 플레이 루프
- 게임 시작/일시정지/재시작/게임오버 처리

## 제외 범위
- 고급 테트리스 규칙(T-spin, hold, ghost piece, SRS 완전 구현)
- 서버 저장 랭킹
- 별도 사운드/이펙트 엔진 추가
