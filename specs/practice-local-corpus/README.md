# README.md

## 기능 배경
- 문장연습이 위키피디아 응답 품질과 네트워크 상태에 따라 문장 난이도와 안정성이 흔들린다.
- 속담/고정 문구는 이미 로컬 데이터로 보유하고 있지만 현재는 폴백으로만 사용된다.

## 목적
- 문장연습 런타임을 로컬 문장 데이터셋 기반으로 고정한다.
- 외부 사전/공공데이터는 배치 수집 파이프라인으로만 사용한다.

## 범위
- `usePracticeText` 로컬 데이터 전용 전환
- `TypingInput` 위키 폴백 UI 제거
- 우리말샘/공공데이터 수집 스크립트 추가

## 제외 범위
- 끝말잇기/단어낙하의 사전 API 사용 방식 변경
- 배포 설정 및 환경별 런타임 인프라 변경

## 성공 기준
- 문장연습 진입 시 네트워크 없이 로컬 문장이 바로 노출된다.
- 데이터셋 갱신은 별도 스크립트로 수행된다.
- lint/test/build 통과.
