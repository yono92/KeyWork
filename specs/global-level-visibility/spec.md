# spec.md

## 문제
1. 플레이어 레벨이 프로필 페이지 안에만 있어 일상 플레이 중에는 거의 보이지 않는다.
2. XP/레벨이 보상 루프로 동작하려면 앱 전반에서 즉시 확인 가능해야 한다.

## 목표
1. 로그인 유저는 사이드바 계정 영역에서 현재 레벨과 진행도를 볼 수 있다.
2. 유저 메뉴 버튼과 드롭다운에서도 현재 레벨을 확인할 수 있다.

## 비목표
- 전역 레벨업 토스트
- 게임 오버 직후 레벨 애니메이션
- 레벨 보상 지급

## 수용 기준
1. 로그인 상태에서 사이드바 계정 영역에 레벨 배지가 표시된다.
2. 유저 메뉴 버튼 또는 메뉴 패널에서 현재 레벨을 볼 수 있다.
3. 비로그인 상태에서는 레벨 UI가 노출되지 않는다.
4. `npm run lint`, `npm run test:run`, `npm run build`가 통과한다.
