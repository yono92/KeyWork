# 커스텀 텍스트 모드 (Custom Text)

## 배경

문장연습 모드는 속담 100개만 제공하여, 숙련 유저는 빠르게 텍스트가 반복됨.
유저가 직접 연습하고 싶은 문장을 등록하면 더 실용적인 연습 가능.

## 목적

유저가 직접 등록한 텍스트로 타이핑 연습할 수 있는 기능 추가.
프로필 또는 연습 화면에서 커스텀 텍스트를 CRUD하고, 문장연습 모드에서 텍스트 소스로 선택.

## 범위

- custom_texts Supabase 테이블
- 커스텀 텍스트 CRUD (등록, 수정, 삭제, 목록)
- 문장연습 모드에서 텍스트 소스 전환 (속담 / 내 텍스트)
- 텍스트 소스 선택 상태 localStorage 저장

## 제외 범위

- 다른 유저의 커스텀 텍스트 열람/공유 (본인만)
- 텍스트 카테고리/태그 분류
- 파일 업로드 (직접 입력만)
