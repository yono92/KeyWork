# spec.md

## 사용자 스토리
- 플레이어로서 문장 연습에서 자연스러운 길이의 문장이 순차적으로 나오길 원한다.

## 기능 요구사항
1. 긴 텍스트를 문장 단위로 분할해 프롬프트 배열로 만든다.
2. 프롬프트 길이는 언어별 상한(한글 90, 영어 140)을 넘지 않는다.
3. Enter 제출 시 다음 프롬프트가 있으면 API 재호출 없이 순환한다.
4. 큐가 비면 다음 텍스트를 새로 가져온다.
5. 괄호/각주 메타 텍스트는 분할 전에 제거한다.

## Acceptance Criteria
1. 긴 텍스트 입력 시 프롬프트가 2개 이상 생성된다.
2. 각 프롬프트 길이는 제한 이내다.
3. 제출 시 다음 프롬프트로 즉시 전환된다.
