# spec.md

## 사용자 스토리
- 플레이어로서, 장애물 단어가 연속으로 너무 비슷하게 나오지 않길 원한다.

## 기능 요구사항
1. 랜덤 단어 선택 시 최근 출제 이력을 고려한다.
2. 한국어는 연속/최근 단어와의 공통 접두를 기준으로 유사 단어를 우선 제외한다.
3. 영어도 동일하게 접두 기준 유사 단어를 우선 제외한다.
4. 후보가 부족하면 기존 단어 풀에서 폴백해 게임 진행이 멈추지 않아야 한다.
5. 게임 재시작 또는 언어 변경 시 최근 이력을 초기화한다.

## 비기능 요구사항
- 기존 성능/렌더링 루프에 영향이 미미해야 한다.
- 기존 타입 안정성과 린트 규칙을 준수해야 한다.

## Acceptance Criteria
1. `TypingRunnerGame`에서 연속 유사 시작어 출제 빈도가 감소한다.
2. `FallingWordsGame`에서 동일한 개선이 적용된다.
3. 공용 훅(`useKoreanWords`)도 동일 규칙으로 동작한다.
4. `npm run lint`, `npm run test:run`, `npm run build` 통과.
