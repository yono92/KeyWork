# spec.md

## 문제
1. 현재 프로필 편집 화면은 카드 한 장에 정보와 편집 기능이 모두 몰려 있어 위계가 약하다.
2. 닉네임 변경과 아바타 편집이 분리되지 않아 화면이 단조롭고 완성도가 낮아 보인다.
3. 성공/실패 메시지, 계정 정보, 현재 프로필 요약이 약해 사용자가 상태를 한눈에 파악하기 어렵다.

## 목표
1. 프로필 페이지를 요약 패널과 편집 작업 영역으로 분리해 화면 구조를 명확히 만든다.
2. 닉네임 변경, 아바타 편집, 계정 정보가 각각 독립된 섹션으로 보이게 한다.
3. 기존 레트로 테마를 유지하면서도 더 밀도 있고 정돈된 UI를 제공한다.

## 비목표
- 인증 로직 변경
- 프로필 데이터 스키마 변경
- 새로운 프로필 기능 추가

## 수용 기준
1. 프로필 페이지가 단일 카드가 아닌 다중 섹션 레이아웃으로 렌더링된다.
2. 닉네임 편집과 아바타 편집은 각각 명확한 액션과 상태를 가진다.
3. 계정 정보와 피드백 메시지가 기존보다 구분감 있게 표시된다.
4. 모바일과 데스크톱에서 모두 레이아웃이 무너지지 않는다.
5. `npm run lint`, `npm run test:run`, `npm run build`가 통과한다.
