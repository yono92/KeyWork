# spec.md

## 문제
1. 일일 미션과 스트릭은 보이지만 장기적인 누적 보상이 없다.
2. 프로필에 현재 성장 상태가 보이지 않아 다시 플레이할 이유가 약하다.

## 목표
1. 점수 이력과 일일 미션 완료를 XP로 환산해 플레이어 레벨을 계산한다.
2. 프로필 상단과 일일 미션 카드에서 현재 레벨, XP 진행도, 오늘 획득 XP를 보여 준다.

## 비목표
- 서버 저장형 시즌 패스
- 별도 보상 수령 버튼
- 전역 레벨업 토스트

## 수용 기준
1. 프로필에서 현재 레벨, 총 XP, 다음 레벨까지 진행도를 볼 수 있다.
2. 일일 미션 카드에서 각 미션의 XP 보상이 표시된다.
3. 오늘 획득한 XP와 오늘 미션으로 얻은 XP가 분리되어 보인다.
4. `npm run lint`, `npm run test:run`, `npm run build`가 통과한다.
