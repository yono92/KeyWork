# spec.md

## 문제
1. 현재 공개 목록형 로비는 방 생성 후 대기 전용 화면에 머물러야 해 답답하다.
2. 사용자는 방에 먼저 들어가 혼자 워밍업하다가 상대가 들어오면 `Ready` 후 시작되는 구조를 원한다.

## 목표
1. 온라인 게임 전부 방 입장 직후 게임룸으로 이동한다.
2. 게임룸에서 상대 대기, `Ready`, 카운트다운, 시작의 공통 플로우를 제공한다.
3. 워밍업 상태와 본게임 상태를 분리해 시작 시 게임 상태를 초기화한다.

## 비목표
- 싱글 플레이 UX 변경
- 랭킹 규칙 변경
- 새로운 게임 모드 추가

## 수용 기준
1. 방 생성/참가 직후 목록 로비가 아니라 해당 게임룸으로 진입한다.
2. 상대가 없으면 게임룸에서 대기 또는 워밍업이 가능하다.
3. 두 플레이어가 모두 `Ready`일 때만 3초 카운트다운 후 시작한다.
4. 테트리스와 끝말잇기 모두 시작 시 워밍업 상태가 초기화된다.
5. `npm run lint`, `npm run test:run`, `npm run build`가 통과한다.
