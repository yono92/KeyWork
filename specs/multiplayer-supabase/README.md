# Supabase 통합: 인증 + 랭킹 + 멀티플레이

## 배경
현재 KeyWork는 완전 싱글플레이어, 로컬 저장만 지원.
유저 계정/랭킹/대전 기능을 추가해서 리텐션과 경쟁 요소를 강화하고 싶음.

## 목적
Supabase를 통합해 3가지 핵심 기능을 구현:
1. **인증** — 소셜 로그인 (Google/GitHub/카카오) + 게스트 플레이
2. **랭킹** — 전 게임 모드 리더보드 (일간/주간/전체)
3. **멀티플레이** — 테트리스 1v1, 끝말잇기 PvP (Realtime)

## 범위
- Supabase Auth (소셜 로그인 + 익명 세션)
- Supabase Database (유저 프로필, 점수 기록, 방 관리)
- Supabase Realtime (대전 동기화)
- 전 게임 모드 랭킹 시스템
- 테트리스/끝말잇기 온라인 대전
- 싱글플레이는 기존과 100% 동일하게 유지 (비로그인도 OK)

## 제외 범위
- 3인 이상 멀티플레이
- 인게임 채팅
- 결제/프리미엄 기능
- 모바일 앱 (웹만)
