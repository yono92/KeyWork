# Spec: Supabase 통합 (인증 + 랭킹 + 멀티플레이)

## 사용자 스토리

### 인증
1. 유저가 Google/GitHub/카카오로 로그인하면 닉네임이 설정되고, 점수가 서버에 저장된다.
2. 비로그인 유저도 모든 싱글 게임을 플레이할 수 있다. 랭킹 등록만 안 됨.
3. 로그인 유저가 다른 기기에서 접속해도 랭킹/설정이 유지된다.

### 랭킹
4. 게임 결과 화면에서 "내 순위"와 TOP 10을 볼 수 있다.
5. 전체/일간/주간 탭으로 기간별 랭킹을 확인할 수 있다.
6. 각 게임 모드별 독립 랭킹이 존재한다.

### 멀티플레이
7. 테트리스/끝말잇기에서 "싱글" / "온라인 대전"을 선택할 수 있다.
8. 방 만들기 → 코드 공유, 또는 빠른 매칭으로 상대를 찾을 수 있다.
9. 테트리스: 라인 클리어 시 상대에게 가비지 라인 전송. 먼저 죽으면 패배.
10. 끝말잇기: AI 대신 상대 유저와 턴제 대전.

## 기능 요구사항

### FR-1: 인증 시스템
- Supabase Auth + `@supabase/ssr` (Next.js App Router 대응)
- **v1: 이메일/비밀번호 가입 + 로그인**
- 회원가입 시 닉네임 설정
- 프로필 페이지: 닉네임 변경, 통계 요약
- 비로그인 유저: 모든 싱글 게임 가능, 랭킹 등록/대전 불가
- 로그인 상태는 헤더에 닉네임 표시
- **TODO (후순위)**: 소셜 로그인 (Google, GitHub, 카카오)

### FR-2: 유저 프로필 & 통계
- `profiles` 테이블: id, nickname, avatar_url, created_at
- `game_scores` 테이블: 게임 결과 기록 (모드, 점수, WPM, 정확도 등)
- 유저별 통계: 총 플레이 수, 최고 점수, 평균 WPM/정확도

### FR-3: 랭킹 시스템
- 게임 모드별 리더보드:
  - **문장연습**: 최고 WPM (정확도 95% 이상만 등록)
  - **단어낙하**: 최고 점수
  - **끝말잇기**: 최고 점수 (싱글 AI 대전)
  - **타이핑 러너**: 최고 거리
  - **테트리스**: 최고 점수
- 기간 필터: 전체 / 이번 주 / 오늘
- 게임 오버 화면에서 랭킹 위젯 표시
- TOP 10 + 내 순위 표시

### FR-4: 로비 & 매칭 (멀티플레이)
- 게임 모드 선택 시 "싱글" / "온라인 대전" 토글
- 온라인 대전 진입:
  - 방 만들기 → 6자리 방 코드 생성 → 대기
  - 방 참가 → 코드 입력
  - 빠른 매칭 → 대기 중인 방에 자동 참가, 없으면 생성
- 2명 입장 시 3초 카운트다운 → 게임 시작
- 로그인 필수 (대전 결과를 기록해야 하므로)

### FR-5: 테트리스 1v1
- 각자 독립 보드 (10×20)
- 상대 보드 미니맵 실시간 표시
- 가비지 라인: 2줄=1줄전송, 3줄=2줄, 4줄=4줄 (1줄 클리어는 전송 없음)
- 가비지 라인: 회색, 랜덤 1칸 빈칸, 보드 아래에서 추가
- 승리/패배 결과 → `game_scores`에 기록 + 랭킹 반영

### FR-6: 끝말잇기 PvP
- AI 로직 → 상대 유저로 교체
- 선공 랜덤. 기존 규칙 동일 (krdict 검증, 두음법칙, 15초 타이머)
- 목숨 3개. 타임아웃/잘못된 단어 시 차감. 상대 목숨 0 = 승리.
- 승리/패배 결과 → `game_scores`에 기록

### FR-7: 연결 관리
- Presence로 온라인 상태 감지
- 상대 5초 오프라인 → 승리 처리
- 새로고침 시 sessionStorage roomId로 재접속 시도

---

## 비기능 요구사항

- 비로그인 싱글 플레이 성능에 영향 없음 (Supabase 코드 lazy load)
- 대전 동기화 지연 300ms 이하
- Supabase 무료 티어 범위 내 동작
- RLS(Row Level Security) 활성화: 유저는 자기 데이터만 수정 가능

## Acceptance Criteria

- [ ] 이메일/비밀번호 회원가입 + 로그인 동작
- [ ] 비로그인 유저가 모든 싱글 게임을 기존과 동일하게 플레이 가능
- [ ] 로그인 유저의 게임 결과가 서버에 저장됨
- [ ] 각 게임 모드별 랭킹 TOP 10 + 내 순위 표시
- [ ] 일간/주간/전체 랭킹 필터 동작
- [ ] 테트리스 1v1 대전: 매칭 → 게임 → 가비지 라인 → 승패 결정
- [ ] 끝말잇기 PvP: 매칭 → 턴제 대전 → 승패 결정
- [ ] 상대 연결 끊김 시 5초 후 승리 처리
- [ ] `npm run build` 통과
