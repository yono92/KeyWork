# spec.md

## 문제
1. 단어낙하에서 단어를 1번 맞췄는데 콤보가 2씩 증가하는 것으로 보이는 현상이 있다.
2. 현재 `handleSubmit`은 동일 단어 중복 처리 방어가 약하고, `setCombo` updater 안에 점수/효과 부수효과가 함께 들어 있다.

## 목표
1. 단어 1회 성공 입력당 콤보가 정확히 1만 증가하게 한다.
2. 같은 단어가 같은 입력 이벤트 흐름에서 중복 처리되지 않게 한다.
3. 점수, 레벨업, 콤보 이펙트는 기존 규칙을 유지하면서 안정적으로 동작하게 한다.

## 비목표
- 단어낙하 난이도 조정
- 아이템 규칙 변경
- 다른 게임 모드의 콤보 시스템 변경

## 수용 기준
1. 동일 단어 1회 성공 입력 시 콤보는 정확히 1 증가한다.
2. 한국어 IME 입력과 일반 Enter 입력 모두 중복 제출로 같은 단어가 두 번 처리되지 않는다.
3. 점수 팝업, 레벨업, 콤보 글로우, 사운드가 기존 의도대로 유지된다.
4. `npm run lint`, `npm run test:run`, `npm run build`가 통과한다.
