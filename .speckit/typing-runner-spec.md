# 타이핑 러너 (Typing Runner) — 현행 스펙

## 개요

무한 러너 + 타이핑 게임. 장애물 위의 단어를 타이핑하면 캐릭터가 점프하여 회피한다. 거리와 속도가 점진적으로 증가한다.

## 유저 스토리

### [P1] US-01: 장애물 단어 타이핑
> 사용자로서, 달려오는 장애물 위의 단어를 타이핑해서 점프로 회피하고 싶다.

**Acceptance Criteria:**
- Given 장애물이 접근 중일 때
- When 장애물 위의 단어를 정확히 입력하면
- Then 캐릭터가 점프하고, 장애물이 제거되며 점수가 가산된다

### [P1] US-02: 거리 기반 난이도 증가
> 사용자로서, 달린 거리에 따라 속도와 난이도가 올라가는 것을 느끼고 싶다.

**Acceptance Criteria:**
- Given 거리가 증가할수록
- When 속도 배율과 단어 길이가 증가하면
- Then 더 어렵고 빠른 게임 플레이가 된다

## 기능 요구사항

### FR-001: 장애물 시스템
- 4종류: 선인장(초록), 바위(회색), 상자(주황), 불꽃(주황/노랑 애니메이션)
- 각 장애물 위에 단어 라벨 표시
- 타겟 장애물의 라벨은 크고 밝게, 비타겟은 작고 어둡게

### FR-002: 난이도 진행
- 스폰 간격: 3000ms 시작, 클리어당 80ms 감소 (최소 1200ms)
- 단어 길이: minLen 2~3, maxLen 3→4→5→6 (클리어 수 기준: 5/15/30회)
- 속도: 클리어당 0.1씩 증가 (초기 속도 2.0)

### FR-003: 점수 시스템
```
클리어 점수 = 단어길이 × 10 + round(현재속도 × 5)
마일스톤: 500m, 1000m, 1500m... 달성 시 축하 애니메이션
콤보: 5회 연속 클리어마다 콤보 카운터 증가
```

### FR-004: 라이프 시스템
- 시작 라이프: 3
- 충돌 시 라이프 -1 + 1.2초 무적 시간 (캐릭터 깜빡임)
- 라이프 0 = 게임 오버

### FR-005: 캐릭터 애니메이션
- 달리기: 2프레임 워크 사이클
- 점프: 팔 올림, 다리 변경
- 피격: 눈 어질, 넉백 애니메이션
- 그림자: 캐릭터 아래 지면 거리 표시

### FR-006: 시각 효과
- 패럴랙스 배경 (3레이어: 원경 산 / 근경 언덕+나무 / 구름)
- 거리 기반 하늘 그라데이션 (낮→밤 사이클)
- 속도 1.5x 이상: 스피드 라인 효과
- 먼지 파티클 (달리기 중)
- 클리어 파티클 + 점수 팝업
- 충돌 시 흔들림 + 줌 + 충격선

### FR-007: UI 구성
- 상단 바 (점수, 거리(m), 속도배율, 라이프, 속도 진행 바)
- 콤보 카운터 (3+: 초록, 5+: 시안, 10+: 노랑)
- 마일스톤 토스트 (거리 달성 축하)
- 하단 입력 필드
