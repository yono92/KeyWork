# 테트리스 (Tetris) — 현행 스펙

## 개요

클래식 테트리스 + 모래 물리 시각 효과. NES/Win95 레트로 픽셀 베벨 스타일. 홀드, 넥스트 프리뷰, 콤보, 10레벨 진행 시스템.

## 유저 스토리

### [P1] US-01: 클래식 테트리스 플레이
> 사용자로서, 떨어지는 블록을 회전/이동하여 줄을 완성하고 점수를 얻고 싶다.

**Acceptance Criteria:**
- Given 블록이 상단에서 낙하 중일 때
- When 방향키로 이동/회전하고 줄을 완성하면
- Then 줄이 클리어되고 점수가 가산된다

### [P1] US-02: 홀드 & 넥스트
> 사용자로서, 현재 블록을 홀드하고 다음 블록을 미리 확인하고 싶다.

**Acceptance Criteria:**
- Given 게임 중 C키를 누르면
- When 현재 블록이 홀드 슬롯으로 이동하면
- Then 이전 홀드 블록(또는 다음 블록)이 활성화되고, 턴당 1회 제한(LOCK 표시)

## 기능 요구사항

### FR-001: 피스 시스템
- 7종 테트로미노: I, O, T, S, Z, J, L
- 각 4회전 상태
- 3D 베벨 셰이딩 (face/hi/lo 색상)

### FR-002: 조작
| 키 | 동작 |
|----|------|
| ← → | 좌우 이동 |
| ↑ | 시계 방향 회전 |
| ↓ | 소프트 드롭 (가속 낙하) |
| Space | 하드 드롭 (즉시 착지 + 충격 효과) |
| C | 홀드 (턴당 1회) |
| P | 일시정지 |

### FR-003: 모바일 조작
- D-pad (좌/우/하 방향)
- 회전 버튼 (↑)
- DROP 버튼 (하드 드롭)
- HOLD 버튼 (사용 후 비활성화)
- 재생/일시정지 토글

### FR-004: 점수 시스템
```
줄 클리어 점수:
- 1줄: 100 × 레벨
- 2줄 (DOUBLE): 300 × 레벨
- 3줄 (TRIPLE): 500 × 레벨
- 4줄 (TETRIS): 800 × 레벨

콤보 보너스 (연속 클리어):
- 콤보 ≥ 1: 50 × 콤보 × 레벨
```

### FR-005: 레벨 시스템
- 레벨 = floor(줄 / 10) + 1, 최대 10
- 드롭 간격 감소:
  - 모바일: max(100ms, 460 - (레벨-1) × 50)
  - 데스크톱: max(140ms, 620 - (레벨-1) × 45)

### FR-006: 고스트 피스
- 착지 위치에 점선 윤곽으로 프리뷰 표시

### FR-007: 애니메이션 & 효과
- 줄 클리어: 흰색 플래시 + "DOUBLE"/"TRIPLE"/"TETRIS!" 라벨
- 하드 드롭: 충격 흔들림 + 임팩트 라인 + 파티클
- TETRIS (4줄): 화면 플래시 (흰색/금색)
- 콤보 ≥ 2: 콤보 표시, ≥ 3: 화면 흔들림
- 모래 물리 오버레이 (Canvas 2D, 랜덤 밝기 그레인)

### FR-008: UI 구성
- 보드: 10 × 20 (3D 베벨 테두리)
- 우측 패널: NEXT 프리뷰, HOLD 프리뷰(LOCK 상태), SCORE/LINES/LEVEL/SPEED
- 콤보 디스플레이
- 조작 범례 (비모바일)
- FX 토글 버튼
- 시작 화면 ("SANDTRIS" 타이틀)
- 일시정지 오버레이 (CONTINUE)
- 게임오버 오버레이 (RETRY)

### FR-009: 캔버스 렌더링
- Canvas 2D 고성능 렌더링
- 배경: #050505
- devicePixelRatio 스케일링
- `image-rendering: pixelated`
