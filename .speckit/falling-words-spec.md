# 단어 낙하 (Falling Words) — 현행 스펙

## 개요

화면 상단에서 떨어지는 단어를 타이핑해서 제거하는 아케이드 게임. 콤보, 파워업, 골든 워드 등 다양한 시스템으로 점수를 극대화한다.

## 유저 스토리

### [P1] US-01: 단어 타이핑으로 제거
> 사용자로서, 떨어지는 단어를 정확히 타이핑해서 제거하고 점수를 얻고 싶다.

**Acceptance Criteria:**
- Given 단어가 화면 상단에서 낙하 중일 때
- When 해당 단어를 정확히 입력하면
- Then 단어가 파티클 폭발과 함께 제거되고 점수가 가산된다

### [P1] US-02: 콤보 시스템
> 사용자로서, 연속으로 단어를 맞추면 콤보 배율이 올라가는 것을 보고 싶다.

**Acceptance Criteria:**
- Given 연속 3회 이상 단어를 맞췄을 때
- When 콤보 카운터를 확인하면
- Then 콤보 수와 배율(최대 2.0x)이 표시되고 시각 효과가 적용된다

### [P1] US-03: 난이도 선택
> 사용자로서, 쉬움/보통/어려움 난이도를 선택하여 내 수준에 맞게 플레이하고 싶다.

## 기능 요구사항

### FR-001: 난이도 시스템
| 난이도 | 스폰 배율 | 속도 배율 | 레벨업 점수 |
|--------|----------|----------|------------|
| Easy | 1.5x (느림) | 0.7x | 400 |
| Normal | 1.0x | 1.0x | 500 |
| Hard | 0.7x (빠름) | 1.3x | 600 |

### FR-002: 레벨 진행
- 레벨업 조건: `level × scorePerLevel` 점수 달성
- 스폰 간격: 2800ms (Lv1) → 600ms (Lv12)
- 낙하 시간: 7.5s (Lv1) → 2.0s (Lv12)
- 레벨업 시 "LEVEL UP!" 애니메이션

### FR-003: 점수 계산
```
일반 단어 점수 = 단어길이 × 10 × 콤보배율 × 래피드보너스
- 콤보배율 = min(1 + combo × 0.2, 2.0)  // 최대 2x (콤보 5+)
- 래피드보너스 = 1.5x (500ms 이내 연속 매치)

골든워드 점수 = 단어길이 × 10 × 3  // 콤보/래피드 미적용, 고정 3배
```

### FR-004: 파워업 (6종)
| 아이템 | 효과 | 조건 |
|--------|------|------|
| ❤️ Life | 라이프 +1 (최대: 기본+2) | — |
| 🐌 Slow | 8초간 전체 감속 (1.5x 낙하시간) | — |
| 💫 Clear | 일반 단어 전체 제거 + 50×레벨 점수 | — |
| 🛡️ Shield | 5초간 미스 1회 보호 | — |
| 💎 Score | 200×레벨 점수 + 콤보 증가 | — |
| ★ Golden | 3배 점수 (황금 단어) | 레벨 3+, 4% 확률 |

### FR-005: 콤보 시스템
- 연속 매치 시 콤보 증가, 미스 시 리셋
- 시각 효과: 3+ 카운터, 5+ 글로우, 10+ 노란색, 15+ 빨간색+화염
- 연쇄 클리어: 1초 이내 연속 매치 시 "DOUBLE!" / "TRIPLE!" / "ULTRA!" / "INSANE!"

### FR-006: 라이프 시스템
- 시작 라이프: 3 (전 난이도)
- 단어가 바닥에 도달 시 라이프 -1
- 라이프 0 = 게임 오버

### FR-007: UI 구성
- 아케이드 스코어바 (점수, 하이스코어, 콤보, 난이도, 레벨, 라이프)
- 활성 효과 인디케이터 (🐌 Slow, 🛡️ Shield)
- 낙하 단어 (일반/아이템/골든 색상 구분)
- 입력 진행 바 (타이핑 진행률)
- 파티클 폭발 효과
- 하단 입력 필드
- 게임오버 모달 (최종 통계)
- 카운트다운 오버레이 (3-2-1)
