# 내 기록 (Local Records) — 현행 스펙

## 개요

글로벌 순위 대신 현재 브라우저에 저장된 5개 싱글플레이 모드의 개인 최고 기록과 최근 기록을 표시한다.

## 유저 스토리

### [P1] US-01: 모드별 기록 조회
> 사용자로서, 각 게임의 최고 점수와 최근 플레이 기록을 확인하고 싶다.

**Acceptance Criteria:**
- Given 내 기록 페이지에 진입했을 때
- When 모드를 선택하면
- Then 해당 모드의 최고 점수와 최근 기록이 최신순으로 표시된다

### [P1] US-02: 저장 범위 인지
> 사용자로서, 기록이 글로벌 순위가 아니라 현재 브라우저에만 저장된다는 사실을 알고 싶다.

## 기능 요구사항

- `keywork.scores.v1`의 로컬 점수만 사용한다.
- 문장연습, 단어 낙하, 끝말잇기, 타이핑 러너, 테트리스 기록을 지원한다.
- 기록이 없으면 빈 상태와 게임 완료 후 저장된다는 안내를 표시한다.
- 계정, 글로벌 순위, 기간별 시즌 필터, 네트워크 요청을 사용하지 않는다.

## 관련 파일

| 파일 | 역할 |
|------|------|
| `app/(game)/leaderboard/page.tsx` | 내 기록 페이지 |
| `src/hooks/useLocalScoreSubmit.ts` | 로컬 점수 제출 |
| `src/lib/localData.ts` | 로컬 점수 저장·조회 |
